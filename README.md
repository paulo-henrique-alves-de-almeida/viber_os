# VibeOS

Sistema operacional inspirado em terminais retrô de tela verde, estética cyber nostálgica e computadores clássicos.

O VibeOS entrega uma experiência imersiva baseada em terminal, misturando minimalismo, atmosfera arcade e interação por comandos.


## Recursos

- Terminal retrô com visual verde monocromático
- Sistema de arquivos simulado
- Aplicativos integrados
- Assistente de IA embutido
- Atmosfera cyber nostálgica
- Suporte a expressões matemáticas
- Comandos secretos


## Comandos Disponíveis

### Sistema

| Comando | Descrição |
|---------|-----------|
| `clear` | Limpa o terminal |
| `whoami` | Mostra o usuário atual |
| `pwd` | Mostra o diretório atual |
| `hostname` | Mostra o nome da máquina |
| `uname` | Mostra informações do sistema |
| `help` | Exibe o menu de ajuda |
| `shutdown` | Desliga o sistema |


### Arquivos e Diretórios

| Comando | Descrição |
|---------|-----------|
| `ls` | Lista arquivos e diretórios |
| `cd [caminho]` | Navega entre diretórios |
| `mkdir [nome]` | Cria diretórios |
| `rmdir [nome]` | Remove diretórios vazios |
| `touch [arquivo]` | Cria arquivos |
| `rm [arquivo]` | Remove arquivos |
| `cat [arquivo]` | Lê o conteúdo de arquivos |
| `echo [texto] > [arquivo]` | Escreve conteúdo em arquivos |


### Inteligência Artificial

| Comando | Descrição |
|---------|-----------|
| `viber [prompt]` | Conversa com Viber, o assistente de IA do VibeOS |


## Aplicativos Integrados

| Aplicativo | Descrição |
|------------|-----------|
| `calendar` | Calendário do sistema |
| `music` | Biblioteca de música |
| `vibegotchi` | Mascote virtual do sistema |
| `vibe_invaders` | Jogo arcade inspirado em clássicos retrô |


## Expressões Matemáticas

O terminal suporta cálculos simples:

```bash
5 + 5
10 * 8
(4 + 2) ** 2
25 % 4
```

### Operadores Suportados

| Operador | Operação |
|----------|----------|
| `+` | Adição |
| `-` | Subtração |
| `*` | Multiplicação |
| `/` | Divisão |
| `//` | Divisão inteira |
| `%` | Módulo |
| `**` | Potência |
| `()` | Parênteses |


## Estética do Sistema

O VibeOS foi criado para capturar a sensação dos antigos terminais monocromáticos e computadores clássicos.

Tela verde. Linhas de comando. Atmosfera silenciosa. Interface minimalista.

Como se um computador esquecido no tempo ainda estivesse ligado em algum lugar.
