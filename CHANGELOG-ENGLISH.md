- New final boss: Vibe Destroyer
- Complete visual overhaul
- New style system: RETRO / VIBE

#### RETRO Style
- Classic v1.0 visuals
- Easy difficulty

#### VIBE Style
- New v1.1 visuals
- Hard difficulty
- New abilities during boss fights

### Achievements
A total of 10 achievements have been added!

- 3 achievements unlocked using VibeOS
- 3 achievements unlocked by playing Vibe Invaders
- 3 achievements unlocked by playing Vibegotchi

## [1.0.0] - 2026-05-11

### Added
- Retro terminal with monochromatic green display
- Simulated file system with support for `ls`, `cd`, `mkdir`, `rmdir`, `touch`, `rm`, `cat` and `echo`
- System commands: `clear`, `whoami`, `pwd`, `hostname`, `uname`, `help`, `shutdown`
- Built-in AI assistant `viber` integrated into the terminal
- Math expression support with operators `+`, `-`, `*`, `/`, `//`, `%`, `**` and `()`
- `calendar` app — system calendar
- `music` app — music library
- `vibegotchi` app — system virtual pet
- `vibe_invaders` game — arcade inspired by retro classics
- Undocumented secret commands waiting to be discovered
- Cyber nostalgic atmosphere based on classic monochromatic terminals
