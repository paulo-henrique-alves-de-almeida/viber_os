# VibeOS

An pseudo-operating system inspired by retro green-screen terminals, nostalgic cyber aesthetics, and classic computers.

VibeOS delivers an immersive terminal-based experience, mixing minimalism, arcade atmosphere, and command-line interaction.


## Features

- Retro monochrome green terminal
- Simulated file system
- Built-in applications
- Integrated AI assistant
- Nostalgic cyber atmosphere
- Mathematical expression support
- Secret commands


## Available Commands

### System

| Command | Description |
|---------|-------------|
| `clear` | Clears the terminal |
| `whoami` | Displays the current user |
| `pwd` | Displays the current directory |
| `hostname` | Displays the machine hostname |
| `uname` | Displays system information |
| `help` | Shows the help menu |
| `shutdown` | Powers off the system |


### Files and Directories

| Command | Description |
|---------|-------------|
| `ls` | Lists files and directories |
| `cd [path]` | Navigates between directories |
| `mkdir [name]` | Creates directories |
| `rmdir [name]` | Removes empty directories |
| `touch [file]` | Creates files |
| `rm [file]` | Removes files |
| `cat [file]` | Reads file contents |
| `echo [text] > [file]` | Writes content into files |


### Artificial Intelligence

| Command | Description |
|---------|-------------|
| `viber [prompt]` | Talks with Viber, the VibeOS AI assistant |


## Built-in Applications

| Application | Description |
|-------------|-------------|
| `calendar` | System calendar |
| `music` | Music library |
| `vibegotchi` | Virtual desktop companion |
| `vibe_invaders` | Retro arcade-inspired game |


## Mathematical Expressions

The terminal supports simple calculations:

```bash
5 + 5
10 * 8
(4 + 2) ** 2
25 % 4
```

### Supported Operators

| Operator | Operation |
|----------|-----------|
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `//` | Integer division |
| `%` | Modulo |
| `**` | Power |
| `()` | Parentheses |


## System Aesthetic

VibeOS was designed to capture the feeling of old monochrome terminals and classic computers.

Green screen. Command lines. Silent atmosphere. Minimal interface.

Like a forgotten machine still running somewhere in time.
